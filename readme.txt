C++ Facebook Project
Welcome to the C++ Facebook project! In this project, you will be using
a command line interface version of Facebook using C++.

Features
- Create new users
- Create new Facebook pages
- Add friends and view your friend list
- Add Pages
- Like and dislike pages
- Post and view status updates
- Save data to a DB file 
- Read data from DB when system initiate

Notes
- The text file will start empty. please add users with the features we've built and then press 12 to save data.
  Then, when reopening the program, the data will be stored from the file straight to the system.

Enjoy :)
